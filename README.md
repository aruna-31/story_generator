** AI Driven StoryTeller** is a premium, modern web application designed to empower creativity through artificial intelligence. Powered by Google's Gemini AI, it allows users to generate immersive stories, design complex game concepts, and explore educational topics in an interactive environment.

## ✨ Key Features

- **📖 AI Story Studio**: Craft unique narratives by defining genres, adding custom characters, and providing plot prompts. Review multiple AI-generated directions and choose the best one for your journey.
- **🎮 Game Design Hub**: Generate detailed game concepts, including platform specifications, core mechanics, and character roles.
- **🎓 Educational Companion**: An interactive AI tutor designed to help students and educators explore complex subjects through natural conversation.
- **👤 Personal Dashboard**: A centralized hub to track your creative history, manage saved projects, and see community highlights.
- **🔒 Secure Authentication**: Robust user sign-up and login system powered by Firebase.
- **🎨 Premium Glassmorphism UI**: A stunning, responsive design featuring vibrant gradients, smooth micro-animations, and a modern aesthetic.
- **🔊 Text-to-Speech Integration**: Bring your stories to life by listening to them directly within the application.

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+ Modules)
- **Styling**: Custom CSS with Glassmorphism effects & TailwindCSS
- **AI Integration**: Google Gemini AI API
- **Backend Service**: Firebase (Authentication & Firestore Database)
- **Build Tool**: Vite

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- [NPM](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aruna-31/story_generator.git
   cd story_generator
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add your API keys:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key
   VITE_FIREBASE_API_KEY=your_firebase_api_key
   # Add other Firebase config values as needed
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open the application:**
   Navigate to `http://localhost:5173` in your browser.

## 📂 Project Structure

- `src/`: Contains core React-like components and logic.
- `firebase.js`: Firebase configuration and service handlers.
- `gemini.js`: Gemini AI API integration and prompt engineering.
- `app.js`: Main application logic and UI orchestration.
- `style.css`: Custom design system and glassmorphism styling.
- `*.html`: Modular pages for Dashboard, Stories, Gaming, and Education.
